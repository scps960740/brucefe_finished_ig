# 布魯斯前端 React仿Instagram教學 完成專案

## 如何使用此專案？

1. 安裝依賴

```
npm i

```

2. 啟動專案

```

npm run start

```

## 如何美化程式碼排版？

```

npm run prettier

```

## 如何啟動Json server？

```

npm run server

```
